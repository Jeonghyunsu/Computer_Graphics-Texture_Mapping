# Texture_Mapping [Computer Graphics]

▶ Texture Files

* Gray scale textures

* RGB textures

* .raw files of 512 x 512 (width x height)

* .raw format doesn’t have any header

-Color image: 3 bytes for each pixel R,G,B value

-Grayscale image: 1 byte for the alpha value
